
## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/loknxq/aquarium_monitor.git
cd aquarium_monitor
```

### 2. Создание и активация виртуального окружения

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS / Linux
python3 -m venv venv
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка базы данных PostgreSQL

Создайте базу данных и пользователя:

```sql
CREATE DATABASE aquarium_db;
CREATE USER your_user WITH PASSWORD 'your_password';
GRANT ALL PRIVILEGES ON DATABASE aquarium_db TO your_user;
```

### 5. Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```env
DATABASE_URL=postgresql+asyncpg://your_user:your_password@localhost:5432/aquarium_db
SECRET_KEY=your_secret_key_min_32_characters
```

### 6. Запуск приложения

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

Приложение будет доступно по адресу: **http://localhost:8000**

### 7. Документация API

- Swagger UI: **http://localhost:8000/docs**
- ReDoc: **http://localhost:8000/redoc**



## Инструкция по использованию

### Регистрация и вход

1. Откройте приложение в браузере (http://localhost:8000)
2. Нажмите кнопку **«Регистрация»**
3. Введите логин и пароль, нажмите **«Продолжить»**
4. После регистрации выполните вход с созданными учетными данными

### Создание аквариума

1. На главной странице нажмите **«Создать аквариум»**
2. Введите название аквариума
3. Выберите тип обитателей (влияет на рекомендации)
4. При желании загрузите фотографию аквариума
5. Отметьте параметры воды, которые планируете отслеживать (температура, pH, нитраты и др.)
6. Нажмите **«Сохранить»**

### Добавление замера

1. На главной странице нажмите на карточку нужного аквариума
2. Перейдите на вкладку **«Добавить замер»**
3. Выберите дату замера (по умолчанию — текущая)
4. Введите значения параметров (отображаются только выбранные при создании)
5. Нажмите **«Сохранить замер»**

### Просмотр истории и графиков

1. На странице аквариума перейдите на вкладку **«История и графики»**
2. При необходимости задайте диапазон дат и нажмите **«Применить»**
3. Для просмотра графика:
   - Отметьте/снимите галочки у параметров, чтобы показать или скрыть линии
   - Наведите курсор на точки графика для просмотра точных значений
4. Таблица с замерами отображается под графиком

### Экспорт данных

1. На вкладке **«История и графики»** нажмите кнопку **«Экспорт CSV»**
2. Файл с замерами автоматически скачается на ваше устройство

### Получение рекомендаций

- Рекомендации отображаются на странице аквариума в верхней части (над вкладками)
- Система анализирует последние замеры и сообщает о параметрах, выходящих за норму

### Личный кабинет

1. Нажмите на кнопку в правом верхнем углу
2. Для смены пароля:
   - Введите текущий пароль
   - Введите новый пароль и его подтверждение
   - Нажмите **«Сменить пароль»**
3. Для выхода из системы нажмите **«Выйти из аккаунта»**

### Удаление аквариума

1. На главной странице найдите нужный аквариум
2. Нажмите на красный крестик в углу карточки
3. Подтвердите удаление

> **Примечание:** При удалении аквариума все связанные замеры также будут удалены без возможности восстановления.
